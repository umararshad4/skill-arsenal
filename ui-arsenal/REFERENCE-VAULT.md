# Reference Vault

**Do NOT pixel-clone.** Identify 3–5 specific design choices (type pairing, spacing rhythm, color discipline, asymmetry, motion) and apply analogous treatment to your project.

## How to use this vault

When you start a surface:

1. **Pick the section** matching the pattern (hero, pricing, dashboard, etc.)
2. **WebFetch one or two** reference sites
3. **Identify 3–5 specific choices** worth emulating:
   - Type pairing & sizing (what fonts, what weights)
   - Spacing rhythm (section padding, element gaps)
   - Color discipline (palette count, where saturation lives)
   - Layout asymmetry (centered vs. off-grid)
   - Motion philosophy (subtle vs. dramatic)
   - Imagery treatment (illustration / photo / abstract / none)
4. **Quote the reference** in your response: *"Modeled after Linear's approach to X — specifically Y."*
5. **Apply analogously** — NOT pixel-copy. The reference shows the *vocabulary*; you compose the sentence.

## Why these specific sites

Each one has been pulled from the same shortlist senior designers cite when asked "show me good." They share traits:

- **Restrained palette** (rarely > 3 brand hues)
- **Distinctive type pairing** (never default sans-on-sans)
- **Deliberate asymmetry** (not centered-by-default)
- **Generous whitespace** (rarely cramped)
- **Subtle motion** (never gratuitous)
- **Real, specific copy** (no marketing-speak filler)

If your work doesn't share these traits, the reference is showing you what to fix.
